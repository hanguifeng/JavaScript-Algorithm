本题思路：

1. 暴力法
   for i = 0; i < n;
   let minheight
   for j = i; j < n;
   minheight = 比较 height[k] -> 最小高度，求得 area
   minheight \* (j - i + 1) 更新 maxArea
   O(n2)
2. 利用栈
   将高度从小到大依次入栈，如果将要入栈的大于栈顶的值，则出栈，直到栈顶值小于要入栈的值，这个值为
   出栈元素的右边界，再算出面积比较，高度为出栈元素的值，宽度为要入栈的元素的下标 - 要出栈元素前一个元素的下标 - 1，栈中初始值为-1(即当左边界没有元素时的处理)
